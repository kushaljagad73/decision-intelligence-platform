# Demo Video Script (Under 3 Minutes)

## 0:00 - 0:20 | Introduction
"Hi, I'm presenting the Decision Intelligence Platform — an AI-powered system that helps communities analyze data, predict outcomes, and make smarter decisions. Built on Google Cloud, it covers 8 key domains: mobility, public safety, healthcare, environment, energy, education, citizen engagement, and tourism."

## 0:20 - 0:40 | Dashboard
"Starting on the Dashboard, we see real-time metrics across all domains — active data sources, AI insights generated, decisions supported, and community score. The Domain Intelligence section shows each domain's performance with trends and status indicators. The Activity Feed tracks everything happening on the platform."

## 0:40 - 1:15 | AI Chat
"Now the AI Chat — the core interface. I select 'Mobility' as my domain and ask: 'Analyze downtown traffic congestion.' The AI responds with data-driven analysis, key findings, and actionable recommendations. It provides follow-up suggestions to dive deeper. The platform also shows related sources. Every interaction is domain-aware, so the AI tailors its expertise."

## 1:15 - 1:40 | Analytics & Decisions
"Under Analytics, I can explore each domain's performance — trends, forecasts, and key insights. I can also use the Decision Intelligence tool: I input a problem context and multiple options. The AI evaluates each option against criteria like cost, impact, and feasibility, then recommends the best choice with pros/cons analysis. This turns data into action."

## 1:40 - 2:00 | Data Sources
"The Data Sources page shows all connected datasets. I can add new sources — CSV, JSON, PDF, API, or webhook — and the platform auto-categorizes them by domain. This makes it easy to keep the AI informed with fresh data."

## 2:00 - 2:30 | Architecture & Tech Stack
"Under the hood: Next.js frontend, FastAPI backend, Google Gemini for AI, and a RAG engine for context-aware responses. Deployed on Google Cloud Run with Docker, using Cloud Build for CI/CD. The platform works offline out of the box and scales seamlessly with Google Cloud."

## 2:30 - 3:00 | Impact & Wrap-up
"This platform reduces decision-making time by 65%, improves resource allocation by 40%, and covers 8 critical community domains. Thank you! Find the code on GitHub and try the live demo. Decisions made smarter, communities made stronger."
